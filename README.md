# Currency-ExchangeMy Currency Converter
This is a simple, responsive currency converter web application built with HTML, CSS, and JavaScript. The application features a user-friendly interface that allows users to select a quantity and currency for conversion.

Features:
Navigation Bar: The top navigation bar includes links to the Home, About Us, and Contact pages for easy access.
Currency Selection: Users can choose a quantity and select from three currencies: Indian Rupee (INR), US Dollar (USD), and Euro (EUR).
Conversion Display: The converted values are displayed in a table format for easy comparison of different currencies.
Responsive Design: The application is styled with CSS and is compatible with various screen sizes.

Technologies Used:
HTML5 for the structure
CSS3 for styling, including the Poppins font for modern typography
JavaScript for dynamic functionality (to be implemented in scripts.js)
